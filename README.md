# Secure Azure VM Deployment with Terraform

## Description
This project demonstrates how to deploy and secure a Linux VM on Microsoft Azure using Terraform.

## Features
- Deployment of Azure VM using Terraform
- SSH authentication using Ed25519 keys
- Network Security Group configuration
- Custom SSH port (2222)
- Azure Blob Storage integration
- Managed Identity authentication

## Technologies
- Terraform
- Azure CLI
- Linux
- SSH
- Azure Storage
